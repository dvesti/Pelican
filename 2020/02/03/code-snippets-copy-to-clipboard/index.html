Копируем код в буфер обмена

Elegant is the only Pelican theme that comes with a “Copy To Clipboard” button on every code snippet.

Try it out

fmt.Println("Hello Clipboard")

We use excellent JavaScript library, clipboard.js for this feature. Your readers do not need Flash for this feature.
